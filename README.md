## 🌟 About The Project

Modern 3D Room is an interactive frontend project built using pure **HTML, CSS, and JavaScript**.

This project creates a realistic 3D room effect using CSS perspective transforms and dynamic mouse movement tracking. The room responds to user pointer movement, creating an immersive and engaging visual experience.

It is designed as a creative UI experiment and portfolio-ready interactive animation project.

## 🧠 How It Works

The 3D room effect is achieved using:

- CSS `perspective()`
- `rotateX()` and `rotateZ()` transforms
- JavaScript pointer movement detection
- Dynamic transform updates based on cursor position

The room rotates smoothly depending on the user’s mouse position to simulate depth.
